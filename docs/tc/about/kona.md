# 看板娘 Kona

## Kona 介紹
Kona 是 Konado 的專屬看板娘，以**活潑可愛、充滿元氣**的少女形象為核心，不僅是品牌視覺的重要符號，更成為使用者與 Konado 之間具親和力的互動載體。


## Kona 形象
Kona 的身影貫穿於 Konado 的全場景中，無論是啟動介面、官網首頁，還是產品文件，都能透過她的形象為品牌注入鮮活感。隨著 Konado 的版本迭代，Kona 的形象也會同步進化——既貼合新版本的設計風格，也在細節打磨中變得更精緻、更具辨識度。

通常在 Konado 的**大版本更新**時，Kona 會推出專屬的全新形象。每一款形象都有獨特的設計巧思，既延續了她的核心氣質，也為產品增添了更多趣味與記憶點。

當然在小版本更新時，Kona 也會有一些小變化，比如換上不同季節的服飾，或是增加一些小配件，讓形象更加豐富多元。

Konado 2.0 版本的 Kona 形象，在細節刻畫上尤為用心，整體造型更顯靈動元氣，將少女的鮮活感拉滿：
- **髮型**：粉紫色短髮輕盈蓬鬆，髮絲弧度自然靈動，一側別著一枚星星造型的髮飾——小巧精緻的設計如同夜空中閃爍的星子，為整體增添了幾分俏皮與夢幻感。
- **服飾**：內搭帶有海軍領的上衣，自帶青春校園氣息；領口處的橙色領結色彩鮮亮，巧妙打破了基礎款的單調。外層疊穿一件淺灰色開衫，柔和的色調既平衡了內搭的亮色，又增添了一份隨性溫暖的氛圍。下裝是一條藍色百褶裙，裙襬隨動作輕晃，盡顯少女的嬌俏靈動。
- **配飾**：襪子設計是本次形象的點睛之筆——主體色彩豐富且帶有漸層效果，一側印有醒目的「KONADO」字母標識（簡潔大方，強化品牌關聯），同時點綴著明快的三角形幾何圖案，與整體活潑風格呼應，讓腿部線條更顯靈動。腳上的白色鞋子簡約百搭，既保證了造型的完整性，又不會喧賓奪主。

作為 Konado 的初代看板娘，Kona（可娜）的 1.0 形象以「青春活力」為核心，充滿夏日清爽感：
- **髮型與五官**：標誌性的紫色雙馬尾飄逸靈動，髮間點綴著粉色小裝飾，俏皮感十足；眼眸如剔透的紅水晶，明亮有神，盡顯朝氣。
- **服飾**：上身是綠白拼色的短款上衣，衣身印有星星圖案，清新又吸睛；搭配一條白色百褶裙，裙襬輕盈，強化了少女的靈動氣質。腰間繫著一枚綠色蝴蝶結，作為細節點綴，讓整體造型更顯嬌俏。
- **鞋履**：腳上的米色涼鞋簡約休閒，既貼合夏日的清爽主題，又讓整體造型更具日常感，拉近與使用者的距離。
